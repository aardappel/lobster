Lobster LSP (Language Server Protocol)

Implementation

The lobster LSP is written in typescript, and runs with node. It calls internally the lobster compiler for the features. > LSP simply uses command line arguments from the lobster runtime.

I want to use this for x editor/IDE

Usually you have to package this LSP in a addon for your preffered IDE. See your IDE/editors documentation on how to integrate this. PRs are always welcome.

You compile the LSP to a final js file to be execute by node with npm webpack.
